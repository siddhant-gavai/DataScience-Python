# Data Science with Python 🐍

A comprehensive collection of Jupyter Notebooks for learning Python from the ground up, specifically tailored for data science enthusiasts.

## 📁 Repository Structure

The notebooks are organized by topic, covering everything from basic syntax to advanced data structures.

### 📓 Notebooks Index

| # | Notebook Name | Description |
|---|---------------|-------------|
| 01 | [pythonBasic01.ipynb](pythonBasic01.ipynb) | Basics of Python syntax and variables. |
| 02 | [String02.ipynb](String02.ipynb) | String manipulation and methods. |
| 03 | [Operator03.ipynb](Operator03.ipynb) | Arithmetic and comparison operators. |
| 04 | [Input04.ipynb](Input04.ipynb) | User input handling in Python. |
| 05 | [Operator_precedence05.ipynb](Operator_precedence05.ipynb) | Order of operations and precedence. |
| 06 | [06_If_Else.ipynb](06_If_Else.ipynb) | Conditional statements (if, elif, else). |
| 07 | [07_Functions.ipynb](07_Functions.ipynb) | Defining and using reusable code blocks. |
| 08 | [08_Match_Case.ipynb](08_Match_Case.ipynb) | Structural pattern matching (Python 3.10+). |
| 09 | [09_String_Formatting.ipynb](09_String_Formatting.ipynb) | `.format()` and f-strings. |
| 10 | [10_Loops.ipynb](10_Loops.ipynb) | `for` and `while` loop iterations. |
| 11 | [11_List_Methods.ipynb](11_List_Methods.ipynb) | Working with Python lists and their methods. |
| 12 | [12_Tuple_Methods.ipynb](12_Tuple_Methods.ipynb) | Exploring immutable sequences (tuples). |

## 🚀 Getting Started

To run these notebooks locally, ensure you have Python and Jupyter installed:

```bash
# Recommended environment setup
pip install notebook
jupyter notebook
```

## 🛠️ Requirements
- Python 3.10+ (Required for Match-Case feature)
- Jupyter Notebook or Jupyter Lab
